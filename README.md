# Sistema de Reserva de Hoteles UPS
Backend desarrollado en FastAPI y MongoDB para la gestión hotelera.
Módulos incluidos: Seguridad, Clientes, Habitaciones, Reservas, Facturación y Reportes.
Las actualziaciones fueron hechas con forme se fueron encontrando errores ademas de cumplir con los modelos especificados en tareas anteriores y se verifica en los commits quien hizo la actualización.
Los primeros commits no tienen identificación devido a que los elaboradores Anthony Pillzia y Erick Lema olvidaron poner esta identificación.
